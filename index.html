<section> Tag

A tag <section> define uma seção em um documento.

<section>
    <h2>Seção de Exemplo</h2>
    <p>Este é um parágrafo dentro de uma seção.</p>
</section>

<h1> Tag

A tag <h1> define o maior cabeçalho (título).

<h1>Este é um cabeçalho H1</h1>

<div> Tag

A tag <div> define uma divisão ou seção em um documento HTML.

<div>Este é um div.</div>
Este é um div.

<img> Tag

A tag <img> define uma imagem em um documento HTML.

<img src="https://www.example.com/imagem.jpg" alt="Descrição da imagem">

<a> Tag

A tag <a> define um hyperlink.

<a href="https://www.example.com">Este é um link</a>
Este é um link

<ul> Tag

A tag <ul> define uma lista não ordenada.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>
  • Item 1
  • Item 2
  • Item 3

<ol> Tag

A tag <ol> define uma lista ordenada.

<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>
  1. Item 1
  2. Item 2
  3. Item 3

<table> Tag

A tag <table> define uma tabela.

<table>
    <tr>
        <th>Cabeçalho 1</th>
        <th>Cabeçalho 2</th>
    </tr>
    <tr>
        <td>Dado 1</td>
        <td>Dado 2</td>
    </tr>
</table>
Cabeçalho 1 Cabeçalho 2
Dado 1 Dado 2

<form> Tag

A tag <form> define um formulário HTML para entrada de usuário.

<form>
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome">
    <input type="submit" value="Enviar">
</form>
